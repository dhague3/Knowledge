Importante

Ne vous enfermez pas dans l'écosystème d'un seul fournisseur d'IA. Les meilleurs modèles d'IA évoluent sans cesse. Notre plateforme est conçue pour la flexibilité, vous permettant de passer à la technologie la plus avancée dès qu'elle est disponible avec un simple changement dans un fichier config.yml dans votre dépôt GitHub.
